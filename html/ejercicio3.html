En esta página el estudiante practicará etiquetas semánticas de texto y tambien algunas etiquetas personalizadas utilizadas en el curso.

Texto insertado y eliminado

Este parrafo contiene texto recientemente agregado y texto que fue eliminado para ilustrar cambios editoriales. Además, se incluyen etiquetas personalizadas: INCS(custom) y DELS(custom).

Abreviaturas, acrónimos y definiciones

El término HTML es un lenguaje de marcado empleado para estructurar documentos que los navegadores pueden interpretar. En contextos informales, se usan acrónimos como ASAP para expresar "lo antes posible".

Lista de definiciones

En HTML existe una estructura especial para listar términos y sus definiciones: <dl>.

HTML
Lenguaje de marcado utilizado para estructurar el contenido en la web.
CSS
Lenguaje de estilos en cascada que define la presentación visual de los documentos HTML.
JavaScript
Lenguaje de programación que permite agregar interactividad y lógica a las páginas web.

Cita en bloque (blockquote)

"El elemento <blockquote> se utiliza para denotar una cita extensa. Suele renderizarse con sangría y separación del texto principal."

Observa cómo el navegador presenta la cita con un formato diferenciado, lo cual mejora la claridad del contenido citado frente al contenido propio.
